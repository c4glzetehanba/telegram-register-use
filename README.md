## 二、Telegram使用攻略

### 2.1 Telegram使用技巧

1. **群组和频道**
   - **群组(Group)**：Telegram群组是Telegram应用中用于多人交流的功能，只需单击“新建群组”并从联系人列表中选择多个联系人即可创建新的群组对话。群组创建者（管理员）可以设置群组规则，管理成员权限，如谁可以邀请新成员、谁可以发送消息、谁可以更改群组信息等，适合共同讨論话题、分享信息或协作工作。群组最多可包含20万名成员。
   - **频道(Channel)**：Telegram频道是一个用于单向信息分发的工具，能够让一个或多个管理员向大量订阅者发送信息，类似于订阅号。与群组不同，频道的订阅者无法相互看到或与管理员以外的成员互动，更适合关注新闻、资源分享和社区动态。Telegram频道的类型还分为公共频道与私人频道，如果是公共频道，任何人都可以搜索和加入，如果是私人的，只有被邀请的人才能加入。

2. **聊天信息加密**
   - **私密聊天**：Telegram提供了“私密聊天”功能，是一种专为隐私保护设计的聊天模式。采用端到端加密，只有通信的两端能够解密和阅读消息内容，确保信息安全，并且支持消息的自动删除功能。
   - **加密通话**：Telegram的语音通话和视频通话也都采用端到端加密。

3. **自定义主题和夜间模式**
   - **自定义主题**：你可以根据个人喜好调整应用的界面，包括自定义主题颜色、聊天背景、聊天字体的大小等等，甚至可以创建自己的主题，或者从网上下载其他用户制作的主题包。
   - **夜间模式**：Telegram提供夜间模式，减少屏幕亮度对眼睛的刺激，适合夜间使用。

4. **机器人（Bots）**
   - Telegram上有大量的第三方机器人，可以帮助你完成许多任务，比如消息提醒、自动化回复、翻译、数据查询、远程控制等。直接在Telegram的搜索栏中输入关键词可以找到感兴趣的机器人，**搜索的时候要注意用俄语、英语去搜，不要用中文**，或者访问专门汇总Telegram机器人的网站添加。

5. **高级搜索功能**
   - 使用高级搜索功能可以找到消息中包含特定关键词的内容。你可以搜索历史聊天记录或者媒体内容等，支持按日期、类型等筛选。此外，还可以利用Telegram Bot进行搜索，例如输入 `@ImageBot` 用于图片搜索，`@PDFBot` 用于文件搜索。

6. **Pin固定消息**
   - 固定消息功能可以将某条重要的消息固定在聊天窗口的顶部。在大型群组中，管理员可以通过固定消息功能发布重要公告或通知，或者分享重要文件、链接，确保这些信息不会遗失，方便新成员查看。

7. **标签管理**
   - 可以为特定消息添加关键字，如 `#工作`、`#重要` 等，以便于分类和检索。要查找特定标签相关的信息，可以在搜索框内输入 `#标签名`，Telegram将自动列出包含该标签的所有消息。

8. **多设备同步**
   - 允许用户在多个设备（手机、平板或电脑）上同时使用同一个账户进行通讯。Telegram支持iOS、Android、Windows、macOS和Linux等多种操作系统。

9. **快捷操作**
   - **@username**：在群组聊天中，使用“@”符号后跟用户名提及某个特定的用户。
   - **@channel_name**：在频道中，使用“@”符号后跟频道名提及或引用某个频道的内容。
   - **消息引用**：长按某条特定消息，然后选择“引用”选项，方便其他参与者理解上下文。

10. **隐藏对话**
    - 如果你不想让某些对话在主聊天界面上显示，可以将其归档（Archive）或隐藏。被隐藏的聊天将进入“归档聊天”区域，可以在需要时通过向下滑动聊天列表进入查看。

### 2.2 使用时的安全与隐私设置

1. **隐藏手机号（核心基础）**
   不允许任何人通过手机号码来找到你，只能通过你分享的链接或者二维码来添加，这是最重要的。
2. **开启两步验证（2FA）**
   作为一个平均每天有3000个以上账号被盗的软件，不开多重验证就相当于把钥匙挂上门。在设置的**隐私与安全**里，设置一个不常用密码，然后把注册邮箱和接收2FA重置密码的邮箱分别设置好（建议使用 iCloud 或 Gmail）。
3. **防止骚扰设置**
   - 最近上线时间和在线状态：设置为**仅联系人**。
   - 谁可以加我进群/频道：设置为**我的联系人**。
   - 语音通话：建议只开联系人，不和陌生人打电话。
4. **敏感内容屏蔽**
   默认是屏蔽的，如需解除屏蔽（显示所有内容），需要在**网页版的Telegram**隐私设置中更改，客户端里是不显示该选项的。
5. **关闭自动下载**
   自动下载媒体建议关掉或者定期清理，不然手机内容会被吃光，或者下载到病毒软件（安卓手机极容易中招）。同时建议在设置里关掉“同步联系人”。

## 三、Telegram注册及使用常见问题

### 3.1 如何注册Telegram账号？
可以通过下载Telegram应用或访问 `web.telegram.org` 来注册。打开应用或网页后，输入手机号码并验证，然后设置用户名和密码即可。

### 3.2 注册Telegram收不到验证码怎么办？
- 检查手机号格式（国家代码 `+86` 不可省略）。
- 如果短信通道拥堵，可以等待官方的倒计时结束，尝试使用“**拨打电话语音验证**”。
- 或者在注册界面检查是否有 Telegram X 或官方邮箱注册选项。

### 3.3 Telegram登录设备限制
Telegram官方仅支持先在**手机端（iOS/Android）**进行初次账号注册。电脑端（网页版或桌面客户端）无法直接注册新账号，必须先在手机注册成功后，再通过扫码登录。

### 3.4 Telegram账号被封禁(+86限制）
由于平台风控，中国大陆的 `+86` 手机号注册的新账号，极易触发垃圾信息风控。遇到此问题，建议联系官方客服机器人 `@SpamBot` 发送申诉解除限制。

### 3.5 如何防止被拉入陌生群组？
依次进入 Settings（设置） → Privacy and Security（隐私与安全） → Groups & Channels（群组与频道），将权限修改为 **My Contacts（仅我的联系人）**，防止被垃圾营销号自动拉群。

### 3.6 单向好友限制（账号被限制沟通）
若提示无法向非好友发送消息（例如显示 `can only send messages to mutual contacts`），通常是因为账号被临时判定为发送垃圾信息的账户。同样需要联系 `@SpamBot` 根据提示进行账号解封。

### 3.7 客户端汉化
Telegram默认不支持中文。可以在应用内搜索官方或权威社区提供的简体中文语言包链接（例如 `t.me/setlanguage/zhcn` 或 `t.me/zh_CN`），点击即可一键汉化。

### 3.8 多端消息不同步
Telegram的云端同步有时会受到本地网络代理的延迟影响。若电脑端或平板端长时间未更新消息，可尝试在设置中退出重新登录，或检查当前使用的代理节点（VPN）是否稳定。
